## Different ways to subset rows and columns



```r
Name <- c("Jon", "Bill", "Maria", "Ben", "Tina")
Age <- c(23, 41, 32, 58, 26)
Location <- c("New York", "Chicago", "New York", "Seattle", "Chicago")

df <- data.frame(Name, Age, Location)
df
```

```
##    Name Age Location
## 1   Jon  23 New York
## 2  Bill  41  Chicago
## 3 Maria  32 New York
## 4   Ben  58  Seattle
## 5  Tina  26  Chicago
```

#### select rows and columns using indexes

```r
df[c(2,4), c(1,3)]
```

```
##   Name Location
## 2 Bill  Chicago
## 4  Ben  Seattle
```
#### select rows and columns using `grep`

```r
df[grep("Chicago", df$Location), ]
```

```
##   Name Age Location
## 2 Bill  41  Chicago
## 5 Tina  26  Chicago
```


```r
df[grep("Age", names(df))]
```

```
##   Age
## 1  23
## 2  41
## 3  32
## 4  58
## 5  26
```

#### select rows with a condiiton

```r
df[df$Age >= 40, ]
```

```
##   Name Age Location
## 2 Bill  41  Chicago
## 4  Ben  58  Seattle
```
